# Auditoria-Acceso-Objetos-WindowsServer
Este proyecto documenta la implementación de auditoría de acceso a objetos en Windows Server 2019 con el objetivo de registrar eventos relacionados con apertura, modificación y eliminación de archivos en una carpeta crítica.
Configuración Realizada
Activación de “Auditar acceso a objetos” en la directiva local.
Configuración de auditoría en la carpeta C:\DatosCriticos.
Registro de eventos 4663 y 4660 en el Visor de Eventos.
Evidencias Incluidas
Capturas de configuración de directiva.
Capturas de configuración de auditoría en carpeta.
Registro de eventos generados.
Informe técnico en PDF.
Resultados
Se verificó el correcto funcionamiento de la auditoría mediante la generación de eventos de acceso y eliminación en el registro de seguridad del sistema.
